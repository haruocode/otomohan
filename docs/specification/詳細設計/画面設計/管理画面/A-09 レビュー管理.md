# A-09 レビュー管理

この画面は、ユーザーが「おともはん」に対して投稿したレビューを**運営が監査・管理・分析するための画面** です。

レビューはサービス品質に直結するため、

- 不適切レビューの削除
- サクラ行為の検出
- おともはん評価の変動監視

など、運用上とても重要な管理領域です。

---

# 1. 画面概要

ユーザーが投稿した「おともはんへのレビュー（評価 + コメント）」を運営が **確認・分析・削除・対応** するための画面。

目的：

- 不適切なレビューの排除（誹謗中傷・個人情報など）
- サクラレビュー・不正レビューの発見
- おともはん評価のモニタリング
- 低評価が急増しているおともはんの早期発見
- カスタマーサポート対応のための証跡把握

---

# 2. レビューの種類と属性

レビューは以下の要素を持つ：

- 評価（★1〜★5）
- レビュー本文（コメント）
- 投稿者ユーザーID（匿名表示）
- 対象おともはんID
- 通話ID（連携）
- 投稿日時
- フラグ（不適切 / 通報対象か）

---

## ② レビュー一覧（テーブル）

| 項目                 | 説明                     |
| -------------------- | ------------------------ |
| レビューID           | `rev_1021`               |
| おともはんID         | `otomo_03`               |
| ユーザーID（匿名化） | `user_***`               |
| 評価                 | ★1〜★5                   |
| コメント             | 冒頭30文字程度           |
| 通話ID               | `call_220`               |
| 投稿日               | 2025/02/01               |
| フラグ               | AI検知 / 通報あり / 正常 |
| 操作                 | 詳細・削除               |

状態表示：

- **赤**：不適切（通報 or AI検知）
- **黄**：低評価レビュー
- **緑**：高評価レビュー
- **灰**：コメントなし

# 5. レビュー詳細画面（モーダル or 新規ページ）

## ① 基本情報

```
レビューID：rev_1021
対象おともはん：otomo_03
投稿者：user_112（匿名）
評価：★2
投稿日：2025/02/01 12:20
```

---

## ② レビュー本文

```
「途中で黙ってしまって気まずかったです」
```

違反ワードが含まれる場合はハイライト：

```
「◯◯ が不快でした」（暴言ワード）
```

---

## ③ 連携データ（通話ログ）

```
通話ID：call_220
通話時間：7分
[ 通話ログを見る ]
```

→ A-04 へ遷移

---

## ④ おともはんの評価履歴（分析）

1週間 / 1ヶ月 / 全期間の評価推移：

```
直近1週間： ★4.6 → ★4.2
直近の低評価：3件
通報：1件（暴言）
```

視覚的に折れ線グラフで表示。

---

## ⑤ 審査・対応（管理）

運営が行う操作：

### ● 不適切レビューの削除

```
[ 削除する ]
（理由入力モーダル）
```

- 誹謗中傷
- 個人情報
- 虚偽レビュー

すべて削除履歴として残す。

---

### ● レビュアーのチェック（不正ユーザー）

ユーザーが悪質なレビューを繰り返す場合：

```
[ レビュアーを警告する ]
[ レビュアーを凍結する ]
```

A-02（ユーザー管理）と連動。

---

### ● おともはんへの注意喚起

必要に応じて：

```
[ おともはんへ注意通知を送る ]
（A-07 通知管理へ跳ぶ）
```

品質管理に有用。

---

## ⑥ 審査履歴

```
2025/02/01 admin_02 「不適切な表現を確認、削除」
2025/02/01 admin_01 「評価急落のため確認済」
```

すべて A-10（操作ログ）に記録。

# 6. AIサポート機能（将来）

レビュー内容は自動分類可能：

- 暴言 / 嫌がらせ
- 性的内容
- ネガティブ感情分析
- スパム
- 不自然な高評価

自動フラグを付けて管理者がチェックできるようにする。

---

# 7. エクスポート機能

- おともはん別のレビュー一覧
- 月次レビュー分析レポート
- 不適切レビュー件数
- 削除履歴付きエクスポート

運営レポート作成に便利。

---

# 8. API 連動

| 用途                 | API                           |
| -------------------- | ----------------------------- |
| レビュー一覧         | GET /admin/reviews            |
| レビュー詳細         | GET /admin/reviews/{id}       |
| レビュー削除         | DELETE /admin/reviews/{id}    |
| レビューにフラグ付与 | POST /admin/reviews/{id}/flag |
| 分析（評価推移）     | GET /admin/reviews/{id}/stats |

通話ログ連携：

```
GET /admin/calls/{callId}
```

---

# 9. セキュリティ

- 重要な個人情報は匿名化して表示
- 管理者操作は A-10（監査ログ）に保存
- レビュー削除は二段階確認
- 不正レビュアー検知は厳重に扱う
