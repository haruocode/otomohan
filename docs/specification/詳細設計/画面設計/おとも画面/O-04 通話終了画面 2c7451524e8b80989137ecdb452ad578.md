# O-04 通話終了画面

**おともはん専用の項目（報酬・通話内容のフィードバック）** を追加した正式版として作成します。

---

# 1. 画面概要

おともはんが通話終了後に以下を確認できる画面：

- 通話情報
- おともはん側の報酬
- 終了理由
- 通話内容の簡易メモ（任意）
- 次の通話を受けるための導線

U-05（ユーザー側）と同じ構成をベースにしつつ、**運営側の役割を考慮した要素が追加**される。

---

# 2. 画面構造（UI）

---

## **ヘッダー**

| 要素 | 内容                                  |
| ---- | ------------------------------------- |
| 左   | なし（または戻る）※自動遷移で問題なし |
| 中央 | 通話終了                              |
| 右   | なし                                  |

---

# **メインコンテンツ**

---

## ① 通話終了メッセージ

中央に大きく：

```
通話が終了しました
```

サブメッセージ：

```
たろう さんとの通話が終了しました
```

プロフィール画像（円形）も U 系と統一。

---

## ② 終了理由

以下のいずれかが表示：

- ユーザーが通話を終了しました
- おともはんが通話を終了しました
- 通信状態が不安定なため切断されました
- 相手がアプリを終了しました

---

## ③ 通話情報カード

黒背景カード + ピンク/ブルーアクセントで統一。

表示項目：

```
通話日：2025/01/30
通話時間：07分32秒
接続開始：21:22
接続終了：21:29
```

---

## ④ 報酬（おともはん側専用）

U 系には無い、おともはんだけの項目。

```
今回の報酬： 700pt
累計報酬　： 43,200pt
```

※ 報酬は

- 1 分あたり固定
- または運営設定のレートに応じて算出
  どちらでも拡張可能。

---

## ⑤ メモ欄（任意機能）

おともはんが次回の対応に向けてメモを残せる欄。

```
通話メモ（任意）
   [テキストエリア]
```

保存ボタンは不要（自動保存 or 画面遷移時保存）

---

## ⑥ 次の行動ボタン（最重要）

### **① 待機画面に戻る（O-01）**

→ 大きいボタン（ブルー/ピンク）

### **② 通話履歴を見る（O-08）**

→ 小さめのサブボタン

---

# 3. モーダル / エラー

- メモ保存失敗時：
  「メモを保存できませんでした」
- 通信断検知：
  RTP 停止イベントを O-04 に引き渡し

---

# 4. API 連動

| 機能             | API                            |
| ---------------- | ------------------------------ |
| 通話履歴取得     | GET /calls/{callId}            |
| 課金・報酬明細   | GET /calls/{callId}/billing    |
| メモ保存（任意） | PUT /otomo/calls/{callId}/memo |
| 終了理由         | call_events から取得           |

---

# 5. 状態遷移

```
O-03 通話中画面
      ↓（終了）
O-04 通話終了（本画面）
      ↓
O-01 待機画面  または  O-08 通話履歴
```

---

# 6. デザイン（U 系との統一）

- ダークモード
- ネオンピンク/ブルーのアクセント
- カードとボタンは丸角が基本
- 報酬カードは U-05 の課金カードと左右対称のデザイン
