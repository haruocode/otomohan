# O-05 収益ダッシュボード

この画面は、おともはん側が **自分の収益をリアルタイムに把握するためのホーム統計機能** です。

将来拡張を前提に、アプリ内の KPI（通話時間／収益／リピート率など）を視覚化する構造にしています。

---

# 1. 画面概要

おともはんが「今までどれだけ稼いだのか」「どの時間帯に通話が多いか」「どのユーザーがリピートしているか」などを確認するための画面。

目的：

- モチベーション向上
- 稼働戦略の最適化
- 今後の運営改善につながる分析データ提供

ユーザー側には提供しない、**おともはん専用のアナリティクス画面**。

---

# 3. メインコンテンツ

---

## ① 期間フィルター（最上段）

```
[本日] [今週] [今月] [全期間]
```

もしくはカレンダーから範囲指定できる UI。

---

## ② 収益サマリー（KPIカード）

カードを横並びまたはGridで表示：

- **総収益：4,320pt**（選択期間中）
- **通話時間合計：120分**
- **平均通話時間：7.5分**
- **リピート率：42%**

視覚的にモチベーションが上がるよう、ネオングラデーションのカードにする。

---

## ③ 収益推移グラフ（折れ線 or 棒グラフ）

例：日別の収益推移（pt）

```
1/1   350pt
1/2   800pt
1/3   0pt
1/4   250pt
...
```

- 棒グラフを縦に並べ、期間比較がしやすい
- タップでその日の「通話時間」「通話数」を表示

---

## ④ 時間帯別通話ヒートマップ（将来対応）

「どの時間帯に通話が多いか」を実感できるヒートマップ。

例：

| 時間帯 | 通話数 |
| --- | --- |
| 6時 | ●○○○○ |
| 12時 | ●●○○○ |
| 18時 | ●●●○○ |
| 22時 | ●●●●○ |

濃いほど通話数が多い。

おともはんが **稼ぎやすい時間帯** を理解できる。

---

## ⑤ 通話一覧（期間内）

```
・たろう さん       7分      100pt
・みゆき さん       20分     300pt
・さくら さん       12分     200pt
```

→ タップで O-09（通話詳細）へ遷移。

---

## ⑥ リピートユーザー統計（任意）

例：

```
リピートユーザー：12人
新規ユーザー：5人
最も話しているユーザー：たろう（合計50分）
```

---

## ⑦ 注意事項（下部）

```
ポイントは参考値です。最終的な報酬は運営の規定に基づき計算されます。
```

# 4. API 連動

| 目的 | API |

| 時間帯別ヒートマップ | GET /otomo/stats/hourly |
| 通話一覧 | GET /otomo/stats/calls |
| リピート統計 | GET /otomo/stats/repeat |

---

# 5. デザイン（トーン&マナー）

- ダークモード + ネオンブルー/ピンク
- KPIカードは光沢のある強調デザイン
- グラフはシンプルで可読性重視
- モチベーションが上がる“未来の機能感”を強調
