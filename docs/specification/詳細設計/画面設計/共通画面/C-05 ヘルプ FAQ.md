# C-05 ヘルプ / FAQ

# 1. 画面概要

ユーザーが疑問・トラブルを自己解決できるように整理された FAQ 画面。よくある質問はカテゴリ別にまとめ、タップで項目を展開する **アコーディオン UI** を採用。

```
■ 通話機能
   ▼ 通話が開始されません

   ▼ 相手の声が聞こえません
      マイク・スピーカーへのアクセス許可をご確認ください。
```

- 「▼」「▲」で開閉
- アニメーションは 150ms 程度

# 4. 代表的な質問と回答（初期セット）

### **【ポイント関連】**

**Q：ポイントの有効期限はありますか？**

A：現時点では無期限です（将来仕様変更の可能性あり）。

**Q：ポイントが反映されません**

A：購入後、反映まで数秒かかることがあります。改善しない場合はお問い合わせください。

---

### **【通話関連（SFU版を考慮）】**

**Q：通話が勝手に切れました**

A：ネットワーク品質が低下した場合、30 秒の無音状態で自動終了する場合があります。

**Q：相手の声が聞こえない**

A：マイク・スピーカーの権限、または端末の音量設定をご確認ください。

**Q：着信に気づけません**

A：通知設定が OFF になっている可能性があります。

設定 → 通知設定 からオンにしてください。

---

### **【アカウント関連】**

**Q：メールアドレスを変更できますか？**

A：変更できません。新規アカウントをご利用ください。

**Q：アカウントを削除したい**

A：設定 → アカウント削除 から行えます。

# 6. デザインルール

- ダークモード基調
- セクションタイトルはピンク/ブルーグラデーションライン
- アコーディオンの枠は薄いネオン
- テキストは白〜グレーで階層を表現
- シンプルで落ち着いた情報 UI
