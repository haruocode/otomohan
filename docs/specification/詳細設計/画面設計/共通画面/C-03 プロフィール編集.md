# C-03 プロフィール編集（名前・アイコン・自己紹介）

## 1. 画面概要

ユーザーが自分のプロフィール情報を編集する画面。変更可能項目は **アイコン画像 / 名前 / 自己紹介文**。メールアドレスはセキュリティ理由により編集不可。

---

# 2. 画面構造

## **ヘッダー**

| 要素 | 内容                     |
| ---- | ------------------------ |
| 左上 | ＜ 戻る                  |
| 中央 | プロフィール編集         |
| 右上 | 保存（アクティブボタン） |

---

# **メインコンテンツ**

## ① プロフィール画像（アイコン）

- 大きな円形のアイコンを中央上に表示
- その下に **「画像を変更する」** ボタン（押下で画像選択UIを表示）
- 画像形式：JPEG / PNG / WEBP
- 最大サイズ：5MB

---

## ② 入力フィールド（フォーム）

### **[名前]**

- ラベル：名前（必須）
- テキスト入力フィールド
- 最大 20 文字
- バリデーション：空不可 / 禁止文字チェック

---

### **[メールアドレス（変更不可）]**

- ラベル：メールアドレス（変更不可）
- グレーアウトされたテキストフィールド
- 入力不可（参照のみ）

---

### **[自己紹介]**

- ラベル：自己紹介
- テキストエリア（3〜5行）
- 最大 300 文字
- 文章例を placeholder に表示
  例：「趣味・得意分野・話せるテーマなどをご記入ください」

---

## ③ サブアクション

画面下部にリンク形式で：

- **パスワードを変更する**

→ 押下でパスワード変更画面（C-04）へ遷移

---

# 3. ボタン仕様

### **保存ボタン（右上）**

- アクティブ：ピンク/ブルーのグラデーション
- 非アクティブ：グレー（入力内容にエラーがある場合）

押下で以下を実行：

1. 入力バリデーション
2. 更新API呼び出し（`PUT /user/profile`）
3. 成功時：マイページ（U-10）へ戻る
4. 失敗時：エラーToast表示

---

# 4. エラー仕様

### 名前欄

- 「名前を入力してください」
- 「使用できない文字が含まれています」

### 自己紹介欄

- 「300文字以内で入力してください」

### アイコンアップロード

- 「対応していない画像形式です」
- 「画像サイズは5MB以内にしてください」

---

# 5. API との関連

| 項目         | API                 |
| ------------ | ------------------- |
| 名前更新     | `PUT /user/profile` |
| 自己紹介更新 | `PUT /user/profile` |
| アイコン更新 | `PUT /user/avatar`  |

※ 保存ボタン押下時、`PUT /user/profile` と `PUT /user/avatar` の両方を実行するケースあり。

---

# 6. 画面遷移

```
U-10 マイページ
      ↓（プロフィール編集ボタン）
C-03 プロフィール編集
      ↓（保存）
U-10 マイページへ戻る
```
