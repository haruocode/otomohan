# C-04 設定画面

# 1. 画面概要

ユーザーアカウント・アプリの利用環境に関する設定を行う画面。

変更頻度は高くないため、**情報整理性・階層構造** が重要。

設定項目は以下を中心とする：

- アカウント（プロフィール / パスワード変更）
- 通知設定
- アプリ設定（テーマ切り替えなど任意）
- サポート
- ログアウト / 退会

---

# **メインコンテンツ**

## ① アカウントセクション

### **（1）プロフィールを編集する**

- 遷移先：C-03 プロフィール編集

### **（2）パスワードを変更する**

- 遷移先：C-05 パスワード変更（予定画面）

### 表示例：

- 「プロフィールを編集」
- 「パスワードを変更」

---

## ② 通知設定（任意）

（仕様が固まり次第 JSON API と連携可能）

項目例：

- プッシュ通知
  - ON / OFF（トグルスイッチ）
- メール通知
  - ON / OFF

バックエンド対応：

- `PUT /user/settings`（通知設定予定 API）

---

## ③ アプリ設定（任意）

項目例：

- テーマ
  - ダークモード（固定）
    ※ 将来ライトモード対応する場合はここで切り替え
- 音声設定
  - 通話中のビープ音 ON / OFF
  - マイクテスト（任意）

---

## ④ サポート

### **（1）利用規約**

- `GET /settings/terms_url` で取得したリンクを表示

### **（2）プライバシーポリシー**

- `GET /settings/privacy_url` リンクを表示

### **（3）お問い合わせ**

- 外部フォームまたはメールリンク

---

## ⑤ アカウント操作

### **（1）ログアウト**

- `POST /auth/logout`
- 成功後：ログイン画面 C-01 へ遷移

### **（2）アカウント削除（任意）**

- `DELETE /user/delete`
- 二段階確認モーダルを挟む

# 4. 各項目の遷移先

| 項目                 | 遷移先               |
| -------------------- | -------------------- |
| プロフィールを編集   | C-03                 |
| パスワードを変更     | C-05（未作成）       |
| 通知設定             | 同画面内のトグル反映 |
| 音声設定             | O-xx（任意画面）     |
| 利用規約             | 外部リンク           |
| プライバシーポリシー | 外部リンク           |
| お問い合わせ         | 外部リンク           |
| ログアウト           | C-01 へ戻る          |
| アカウント削除       | モーダル → 削除実行  |

# 5. API 連携

| 機能                          | API                   |
| ----------------------------- | --------------------- |
| 通知設定変更                  | `PUT /user/settings`  |
| プロフィール編集              | `PUT /user/profile`   |
| パスワード変更                | `PUT /user/password`  |
| ログアウト                    | `POST /auth/logout`   |
| 退会                          | `DELETE /user/delete` |
| 利用規約・プライバシーURL取得 | `GET /settings`       |
